4. Key Behaviors

BOOT
First visit: condensed boot. Return visits: skip. localStorage flag.
WINDOWS
Fullscreen with title bar (traffic lights + ◀ back). No drag/resize.
BOTTOM NAV
3 bordered tab buttons (C1 style). Persistent on all screens. Active glows.
HIDDEN
Schedule widget, sponsor logos, desktop dock, resize handles.
