3. Key Behaviors

BOOT SEQUENCE
First visit: condensed 2-3s boot animation. Return visits: skip straight to home screen. Tracked via localStorage.
WINDOW CHROME
Title bar with traffic light dots + ◀ back button. No drag, no resize. Close = back to previous screen.
BOTTOM NAV
Persistent C1 bordered tab buttons. 4 tabs: Home, Bounties, Events, User. Active tab glows. Visible on all screens.
HIDDEN ON MOBILE
Schedule widget, sponsor logos, desktop dock, drag/resize handles. All desktop-only.
